Vinea is seeking integration with AWIN's UK supermarket and wine merchant affiliate programs to convert AI-powered wine recommendations into sales.

The Vinea Wine Engine

Vinea is an iOS application that learns its users preferences and helps them discover styles of wine that match their palate. It matches user taste profiles to styles of wine, and allows users to find the perfect style for their meals.

Taste profile exploration

Users wine reviews create a taste profile, which is used to recommend new styles to try.

AI-powered matches

Users input their evening meal. Vinea's AI acts as a digital sommelier, taking into account the user's preferences to recommend the exact wine structure and style required for the meal.

Label scanner

Using Apple's Vision framework, users scan physical bottles. Vinea matches the label against the global LWIN database, adding it to their dynamic taste profile.

The proposed affiliate workflow

The user interface helps users find the style of wine, we now need to find the bottle.

Currently the "find" button generates a google search URL string, so users can search their local supermarkets. We want to take them directly to the product.

1
Discovery & AI Recommendation
User explores styles, or receives a recommendation based on their taste profile or specific food pairing.
2
Merchant Selection
Affiliate data is displayed under each style (e.g. when clicking on Red Bordeaux they would see which of these wines are available at Waitrose).
3
One-Tap "Find" Routing
Tapping the "Find" button drops the user directly onto the product page of their affiliate retailer.
